F29AI1 - Lab week 5
Route finding in Java

Introduction

This lab looks at a search program for finding routes in a toy rail map.

 
 

RailSearch class

import java.util.*;    
public class RailSearch {      
 String links[][] = {  {"Edinburgh", "Berwick"},            
                            {"Edinburgh", "Carlisle"},            
                            {"Carlisle", "Lancaster"},            
                            {"Lancaster", "Manchester"},            
                            {"Manchester", "Leeds"},            
                            {"Leeds", "Sheffield"},            
                            {"Carlisle", "Leeds"},  
                            {“Leeds”, “Newcastle”},          
                            {"Sheffield", "Derby"},  
                            {“Sheffield”, “Newcastle”),          
                            {"Derby", "Doncaster"},            
                            {"Doncaster", "Newcastle"},            
                            {"Newcastle", "Berwick"}            };      
 
                            String goal = "Doncaster";

Successor function

                               String goal = "Doncaster";
                               Vector getSuccessors (String location)
                               {
                                                             Vector successors = new Vector ();
                                                             for (int i = 0; i < links.length; i++)
                                                             {
                                                                                            if (links[i][0].equals (location))
                                                                                            {
                                                                                                                          successors.add (links[i][1]);
                                                                                            }
                                                                                            if (links[i][1].equals (location))
                                                                                            {
                                                                                                                          successors.add (links[i][0]);
                                                                                            }
                                                             }
                                                             return (successors);
                               }
                               @SuppressWarnings(value = "unchecked")

Main loop

   void run ()
                                                             {
                                                                                            Vector<Vector> open = new Vector <Vector>();
                                                                                            Vector path = new Vector ();
                                                                                            path.add ("Edinburgh");
                                                                                            open.add (path);
 
                                                                                            while (open.size () > 0)
                                                                                            {
                                                                                                                          path = open.remove (0);
                                                                                                                          String parent =  path.elementAt (path.size () - 1);
                                                                                                                          if (parent.equals (goal))
                                                                                                                          {
                                                                                                                                                         System.out.println ("ROUTE: " + path);
                                                                                                                          }
                                                                                                                          Vector successors = getSuccessors (parent);
                                                                                                                          for (int i = 0; i < successors.size (); i++)
                                                                                                                          {
                                                                                                                                                         String child = successors.get (i);
                                                                                                                                                         if (!path.contains (child))
                                                                                                                                                         {
                                                                                                                                                                                       Vector newPath = (Vector) path.clone ();
                                                                                                                                                                                       newPath.add (child);
                                                                                                                                                                                       open.add (newPath);
                                                                                                                                                         }
                                                                                                                          }
                                                                                            }
                                                             }

main method

      public static void main(String args[]) { // do the search         new RailSearch().run();      }   }

Running the program the traditional way

Compile the program with 'javac', then run it with 'java' command:

   javac RailSearch.java   java RailSearch

Output generated

The program generates all possible routes connecting `Edinburgh' with `Doncaster', ordered by length.

Questions

  • Does the ordering of link data make any difference? Does it make any difference if {'Derby', 'Newcastle'} comes before {'Derby', 'Doncaster'} in the array?
  • Does it make any difference if {'Derby', 'Doncaster'} appears as {'Doncaster', 'Derby'}?
  • How is the goal for a particular search presented? How could we improve on this?
  • What does the Vector 'remove' method do? Suggest a more meaningful name for this method.
  • What search strategy does this program use for purposes of producing solution paths?

Exercises

·  Modify the program so that it produces only the shortest route between two locations.

·  Modify the program so that it prints out the total number of locations inspected in order to obtain the shortest route between two locations.

·  Modify the program so that it performs depth-first search if the value of a boolean class variable 'dfs' is <true>, and breadth-first search otherwise.

·  Modify the program so that it searches to a maximum depth of four levels in the search tree.

·  Modify the program so that the depth-limit may be passed in as a command-line argument. (This will involve modifying the 'main' method.)
